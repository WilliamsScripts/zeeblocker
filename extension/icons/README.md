# Icons

To generate icons for ZeeBlocker:

1. Open `generate-icons.html` in a web browser
2. The icons will be automatically generated and downloaded
3. Place the downloaded `icon16.png`, `icon48.png`, and `icon128.png` files in this directory

Alternatively, you can use any image editing software to create 16x16, 48x48, and 128x128 pixel icons with the ZeeBlocker branding (purple gradient with a shield/block symbol).
