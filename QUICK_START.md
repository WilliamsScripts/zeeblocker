# ⚡ ZeeBlocker Quick Start Guide

Get up and running with ZeeBlocker in 5 minutes!

## 🚀 Installation (2 minutes)

### Step 1: Prepare Icons
1. Open `icons/generate-icons.html` in Chrome
2. Three icon files will download automatically
3. Move them to the `icons/` folder

### Step 2: Install Extension
1. Open Chrome and go to `chrome://extensions/`
2. Enable **Developer mode** (top right toggle)
3. Click **Load unpacked**
4. Select the `zeeblocker` folder
5. Done! 🎉

## 🎯 Using Focus Mode (30 seconds)

1. Click the ZeeBlocker icon (🎯) in your toolbar
2. Toggle **Focus Mode** to ON
3. Try visiting Facebook, Twitter, or YouTube
4. You'll see a block page! ✅

**That's it!** You're now blocking distracting sites.

## ✍️ Adding Your First Task (1 minute)

1. Click ZeeBlocker icon
2. Click **+ Add Task**
3. Enter:
   - Task title: "Finish report"
   - Start time: "09:00"
   - End time: "11:00"
   - Description: "Complete Q4 report"
4. Click **Save**

Your task is now tracked! ✅

## ⚙️ Essential Settings (1 minute)

Click the **⚙️ (Settings)** button in the popup:

### Customize Your Blocklist
1. Go to **Blocklist** tab
2. Enter a site (e.g., "linkedin.com")
3. Click **Add Site**

### Enable Dark Mode
1. Stay on **General** tab
2. Toggle **Dark Mode** ON
3. Everything turns dark! 🌙

## 👨‍👩‍👧 For Parents

### Quick Child Safety Setup
1. Open Settings → **Child Safety** tab
2. Toggle **Enable Child Safety Mode** ON
3. Enter your email
4. Done! You'll be notified of restricted site attempts

⚠️ **Important**: Discuss internet safety with your child and ensure you have consent to monitor.

## 🏢 For Organizations

### Quick Organization Setup
1. Install on all employee laptops (same steps as above)
2. Open Settings → **Organization** tab
3. Toggle **Enable Organization Mode** ON
4. Enter admin email
5. Inform employees about the policy (required!)

## 🔗 Connecting Integrations

### Jira Example
1. Settings → **Integrations** tab
2. Toggle **Jira** ON
3. Enter your Jira domain (e.g., `company.atlassian.net`)
4. Get API key from Jira settings
5. Paste API key
6. Click **Test Connection**
7. Click **Sync All Integrations**

Tasks from Jira now appear in ZeeBlocker! ✅

## 📊 Viewing Your Stats

Click the ZeeBlocker icon to see:
- **Blocked Today**: Sites blocked count
- **Focus Time**: Time spent in focus mode
- **Tasks Done**: Completed tasks

## 🌙 Dark Mode Everywhere

1. Click 🌙 icon in popup header
2. OR go to Settings → General → Dark Mode
3. Entire extension switches to dark theme

## 🆘 Common Questions

**Q: Sites aren't being blocked?**  
A: Make sure Focus Mode is ON and the site is in your blocklist.

**Q: How do I stop blocking?**  
A: Toggle Focus Mode OFF in the popup.

**Q: Can I block YouTube but not specific channels?**  
A: Currently blocks entire domains only. Fine-grained control coming soon!

**Q: Does this work in Incognito?**  
A: Only if you enable it in `chrome://extensions/` → Details → Allow in Incognito.

**Q: How do I uninstall?**  
A: `chrome://extensions/` → Find ZeeBlocker → Remove.

## 💡 Pro Tips

### Tip 1: Custom Time Slots
Create tasks for your entire day to stay organized:
- 09:00-10:00: Email & planning
- 10:00-12:00: Deep work
- 13:00-15:00: Meetings
- 15:00-17:00: Project work

### Tip 2: Strategic Blocking
Don't block everything! Keep sites you need for work but block only the distracting ones.

### Tip 3: Use Idle Alerts
Set idle time to 15 minutes to remind yourself to take breaks.

### Tip 4: Share Your Blocklist
Export your settings and share with teammates for consistency.

### Tip 5: Review Stats Weekly
Check your blocked count and focus time to track improvement.
